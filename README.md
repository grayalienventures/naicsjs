# naics
Utility tools for 2022 NAICS standards.

## Installation
Install this package using npm.
```
$ npm install naicsjs
```

## Usage
Get industry by NAICS code.
```javascript
const naics = require('naicsjs')

console.log(naics['62111'])
```
